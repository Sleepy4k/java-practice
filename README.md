# Java Practice

Learn basic syntax for java

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/Sleepy4k/java-practice.git
~~~

Go to the project directory  

~~~bash  
  cd java-practice
~~~

Build the java file into class

~~~bash  
  javac <filename>.java
~~~

Start the java class  

~~~bash
  java <filename>
~~~

## Usage/Examples

~~~bash
  javac hello.java
~~~  

~~~bash
  java hello
~~~
